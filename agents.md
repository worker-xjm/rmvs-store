## AI 协作规范（Agents）

本文件用于约束 / 提示在本项目中协作的 AI Agent 行为。后续你可以继续在本文件中追加更多规则和提示词。

### 样式相关硬性要求

- **唯一硬性要求**：在实现或修改任何页面、组件、布局时，**样式必须使用 Tailwind CSS 4.x**。
- 使用的 Tailwind 版本需与项目当前配置保持一致（参见 `astro.config.mjs` 中的 Tailwind 集成）。
- 不得引入其他 CSS 框架（如 Bootstrap、MUI 等）替代 Tailwind；如确有需要，必须显式在本文件中新增规则说明。
- 如需编写额外 CSS，请以 Tailwind 原子类为主，仅在 `src/styles/global.css` 中补充少量无法通过原子类优雅表达的全局样式。

### 开发协作说明

- 在创建或修改组件时，请优先思考如何通过 Tailwind CSS 4.x 完成排版与视觉表现。
- 保持类名语义清晰、可维护，避免无意义的重复样式。
- 如有不确定的 Tailwind 写法，优先查阅官方文档，并在提交前保持类名简洁。

> 说明：目前本文件只包含「**样式必须使用 Tailwind CSS 4.x**」这一主要约束。后续如有新的协作提示或规则，会在下方继续追加。
